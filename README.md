# 🐍 Smart Code Error Helper

## 📌 Project Overview
The Smart Code Error Helper is a Python-based tool designed to help beginners understand and debug their Python code. Instead of only showing error messages, this tool analyzes the user's code, identifies errors, and provides clear explanations along with possible solutions.

---

## ❗ Problem Statement
Beginner programmers often struggle to understand Python error messages. While Python provides error types and line numbers, the explanations are not always easy to understand, making debugging difficult.

---

## 💡 Solution
This project allows users to:
- Paste their Python code
- Automatically detect errors
- Identify the exact line where the error occurred
- Provide a simple explanation of the error
- Suggest possible solutions to fix it

---

## ⚙️ Features
- Accepts multi-line Python code input
- Detects both syntax and runtime errors
- Displays error line number
- Provides clear explanation of errors
- Suggests solutions for fixing errors
- Simple menu-driven interface

---

## 🛠️ Technologies Used
- Python
- Exception Handling (`try-except`)
- `traceback` module
